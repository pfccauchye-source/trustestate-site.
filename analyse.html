🗂️ Dossier de due diligence — documents du bien

Déposez le DPE officiel, les diagnostics (amiante, plomb, termites…), le compromis de vente ou les PV de copropriété. L'IA les lit et les croise avec l'analyse en données ouvertes ci-dessus : cohérences, écarts, informations nouvelles. Rien n'est conservé : les fichiers sont traités dans votre navigateur puis le texte est envoyé une seule fois pour l'analyse.

Méthode (TrustScore, transparente). Note globale sur 100 pondérée sur quatre axes : Valeur (30 % — positionnement prix vs ventes réelles DVF + liquidité), Risque (25 % — risques naturels Géorisques + risque réglementaire DPE), Potentiel (25 % — tendance des prix DVF + dynamique) et Qualité (20 % — commodités du quartier : transports, écoles, commerces, santé, via OpenStreetMap). Les axes affichent « estimation » lorsque les données sous-jacentes sont trop rares. Les montants affichés (rénovation, écart de prix) sont indicatifs, avec hypothèses précisées.
